Functions

spawn_entity

The function spawn_entity spawns an entity, and it takes 7 parameters. They are as follows:

  id: The ID of the entity that you spawn.

  x: The X position of the spawned entity.

  y: The Y position of the spawned entity.

  s: True for screen coordinates and false for coordinates relative to the player.

  vx: The velocity on the X axis of the spawned entity.

  vy: The velocity on the Y axis of the spawned entity.

  snap: If the entity snaps to the auto grid or not. Must be true or false, respectively.

Example:

spawn_entity(631, 0, 0, false, 0, 0, true) -- Spawns an big explosion on the player when they enter a level

spawn_door

The function spawn_door spawns a door, and it takes 6 parameters.

  x: The X position of the door.

  y: The Y position of the door.

  l: The level of the world that the door links to.

  w: The world the door links to.

  t: The theme of the level the door links to. Can be any of the following options:

  • THEME.DWELLING
  • THEME.JUNGLE
  • THEME.VOLCANA
  • THEME.OLMEC
  • THEME.TIDE_POOL
  • THEME.TEMPLE
  • THEME.ICE_CAVES
  • THEME.NEO_BABYLON
  • THEME.SUNKEN_CITY
  • THEME.COSMIC_OCEAN
  • THEME.CITY_OF_GOLD
  • THEME.DUAT
  • THEME.ABZU
  • THEME.TIAMAT
  • THEME.EGGPLANT_WORLD
  • THEME.HUNDUN
  • THEME.BASE_CAMP

onLevel

The function onLevel() cannot be called, and is automatically called at the start of each level. Example:

function onLevel()
    spawn_entity(632, 0, 0, false, 0, 0, true) -- Spawns an explosion on the player
end

onFrame

onFrame executes every frame. It is similar to onLevel, as it cannot be called directly.

function onFrame()
    spawn_entity(632, 0, 0, false, 0, 0, true) -- Spawns an bomb on the player every frame - will crash your game
end
