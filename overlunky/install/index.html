Development

If you want to help with development, get cmake and git, and then type the following into a command prompt:

git clone https://github.com/spelunky-fyi/overlunky/
cd overlunky
git submodule update --init --recursive
mkdir build
cd build
cmake ..
cmake --build . --config Release --target ALL_BUILD

Lua Support

Since the main page does not have Lua support, the command is slightly different.

git clone https://github.com/Dregu/overlunky/
cd overlunky
git submodule update --init --recursive
mkdir build
cd build
cmake ..
cmake --build . --config Release --target ALL_BUILD
